MIMIC

MIMIC is an open-source session replay framework which simplifies recording & replaying user sessions to aid debugging web applications

For the developer, MIMIC provides a Python CLI installer to instrument applications, a complete backend infrastructure, and fully-loaded frontend dashboard to get to the "why" and "how" around user experienced errors with ease.
